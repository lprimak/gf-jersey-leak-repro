== How to debug
* Open the Jersey project in your IDE
* Check out 4.0.2-RELEASE tag
* Attach remote debugger to port 9009
* Set breakpoints in the Jersey code: `CdiComponentProvider.java` lines 271 and 725
* You will see the following sequence:
** `effectiveInjectionManager.set()`
** `effectiveInjectionManager.remove()`
** `effectiveInjectionManager.set()`

[NOTE]
====
The bug is that the last `set()` is not followed by `remove()`
====
